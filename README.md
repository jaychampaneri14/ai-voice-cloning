# Ai Voice Cloning

> Text-to-speech with multiple voice styles using gTTS and pyttsx3

## Overview
This project is part of a comprehensive AI/ML portfolio covering python tts.

## Features
- Production-ready implementation
- Well-documented code with examples
- Easy to customize and extend

## Installation

```bash
git clone https://github.com/jaychampaneri14/ai-voice-cloning.git
cd ai-voice-cloning
pip install -r requirements.txt
python main.py
```

## Requirements
See `requirements.txt` for full dependencies.
Main libraries: Python, Tts, Audio

## Usage
```bash
python main.py
```
Run with `--help` for full options.
